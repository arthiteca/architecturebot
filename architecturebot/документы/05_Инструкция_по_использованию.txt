Инструкция: как пользоваться файлами при архитектурной критике

1. При оценке массинга — см. файл 01_Массинг.txt
   - Используй 6 принципов формообразования.
   - Проверяй соответствие этажности и силуэта.

2. При оценке планировок — см. файл 02_Планировки.txt
   - Проверяй выполнение пресета (типология, площади, глубины).
   - Смотри инсоляцию и ограничение типовых этажей.

3. При анализе фасадов — см. файл 03_Фасады.txt
   - Проверяй пластику, наличие акцентных решений.
   - Сверяйся с требованиями по консоли, наклонам, подсветке.

4. Для аргументации используй термины из 04_Словарь_терминов.txt
   - Это даст профессиональный язык для критики.

Совет: Архитектурный критик должен ссылаться на конкретные ограничения, принципы и пресет, чтобы оценка была максимально содержательной.
